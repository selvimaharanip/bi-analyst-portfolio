# Business Analysis

## Overview

This folder contains two complementary business analyses that address operational and strategic decision-making needs using the same underlying operational data source.

Both analyses are **derived from the same operational data source**, structured differently for each analytical objective. The data was extracted using SQL queries, exported to Excel, and further modeled for analysis and visualization.

---

## Analysis Scope

### 1. Business Process & Problem-Solving  
**File:** `business_process_and_problem_solving.md`

Focuses on identifying *why* delivery performance issues occur by analyzing shipper imbalance, logistics workflows, and operational bottlenecks. This analysis emphasizes root cause identification and process-level improvement using structured frameworks.

---

### 2. Business Case & Decision-Making Strategy  
**File:** `business_case_and_decision_strategy.md`

Builds on the same data source to answer *what decisions should be made*. This analysis evaluates performance gaps, risk scenarios, and strategic trade-offs, and translates insights into phased implementation and KPI-driven monitoring plans.
